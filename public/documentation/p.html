KittleP Documentation

KittleP is a custom web component for stylized paragraph elements.

Available Attributes

Attribute Description Example
f-sz Sets the font size (e.g., px, em, small). f-sz="18px"
color Sets the text color. color="red"
bg-color Sets the background color. bg-color="lightgray"
rounded Applies border-radius for rounded corners. rounded="8px"
pd Sets padding inside the paragraph. pd="10px"
mg Sets margin outside the paragraph. mg="10px"
content-position Aligns text within the paragraph (left, center, right). content-position="center"
display Defines the display type (block, inline-block, etc.). display="block"
type Defines a special paragraph type (e.g., quotes). type="quotes"

Usage Example

Include the kittle.min.js script and use the component:

<kittle-p f-sz="18px" color="blue" bg-color="lightgray" rounded="8px" pd="10px" mg="10px"> This is a custom paragraph. </kittle-p> <kittle-p type="quotes" color="green" f-sz="20px"> "This is a quote styled paragraph." </kittle-p>
