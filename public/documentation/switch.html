kittle-switch is a lightweight, customizable switch component.

Usage Example

<kittle-switch checked color="#ff5722" size="large" rounded></kittle-switch>

Attributes & Customization

Attribute Description Default
checked Sets the switch to ON state. false
color Sets the color of the switch when ON. #007AFF
size Size options: small, medium, large. medium
rounded Applies rounded corners. false
disabled Disables the switch interaction. false
blur Adds a blur effect to the background. false

JavaScript API

const switchElement = document.querySelector("kittle-switch");
switchElement.addEventListener("change", (event) => {
    console.log("Switch State:", event.detail);
});
