Kittle Table Documentation

<kittle-table> is a custom web component for creating flexible and stylish tables. It allows you to customize table appearance using attributes and insert dynamic table content inside.

Attributes

Attribute Description Example
title Sets the table title title="User List"
bg-color Changes the table background color bg-color="lightgray"
header-color Changes the table header background color header-color="blue"
color Sets the text color color="black"
rounded Sets table border radius rounded="10px"
mg Sets margin mg="10px"
pd Sets padding pd="10px"
line Sets border thickness between rows line="2px"
line-color Sets border color line-color="gray"
blur Adds a blur effect blur="5%"
opacity Controls table transparency opacity="80%"

Usage Example

<kittle-table title="User List" bg-color="white" header-color="blue" color="black" rounded="10px">
    <thead>
        <tr>
            <th>Name</th>
            <th>Age</th>
            <th>Country</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>John Doe</td>
            <td>28</td>
            <td>USA</td>
        </tr>
        <tr>
            <td>Jane Smith</td>
            <td>32</td>
            <td>Canada</td>
        </tr>
    </tbody>
</kittle-table>
