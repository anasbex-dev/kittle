Kittle Animation V1 Documentation

KittleAnim adalah library animasi berbasis JavaScript yang memudahkan pembuatan animasi elemen HTML.

📌 Cara Penggunaan

Tambahkan kittle.min.js ke proyek Anda dan panggil metode yang tersedia.

const anim = KittleAnim("#element").to({ transform: "translateX(100px)" }, 1000);

🛠 Metode & Parameter

1️⃣ to(properties, duration, easing, delay, onComplete)

Menjalankan animasi dari kondisi sekarang ke kondisi yang ditentukan.

  • properties: Object CSS yang akan dianimasikan
  • duration: Durasi animasi (ms)
  • easing: Jenis easing ("linear", "ease-in", "ease-out", dsb.)
  • delay: Delay sebelum animasi dimulai (ms)
  • onComplete: Callback setelah animasi selesai

🔹 Contoh

2️⃣ from(properties, duration, easing, delay)

Menjalankan animasi dari kondisi awal yang ditentukan ke kondisi default elemen.

🔹 Contoh

3️⃣ fromTo(fromProps, toProps, duration, easing, delay)

Menjalankan animasi dari kondisi awal ke kondisi akhir.

🔹 Contoh

4️⃣ bezier(properties, duration, bezier, delay)

Animasi dengan custom cubic-bezier easing.

🔹 Contoh

5️⃣ spring(properties, stiffness, damping)

Animasi dengan efek pegas.

🔹 Contoh

6️⃣ path(pathString, duration, easing)

Animasi mengikuti path SVG.

🔹 Contoh

7️⃣ pause() & resume()

Pause dan lanjutkan animasi.
