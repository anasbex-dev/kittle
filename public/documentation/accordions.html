kittle-accordions Documentation

<kittle-accordions> is a customizable web component for creating collapsible accordion sections.

Usage

Include the component in your HTML:

<kittle-accordions title="Click to Open"> <p>This is the content inside the accordion.</p> </kittle-accordions>

Available Attributes

Attribute Type Default Description
title String - Sets the accordion header title.
color String #000 Text color.
bg-color String #fff Background color of the accordion.
border-color String #ddd Border color of the accordion.
header-bg String #f1f1f1 Background color of the header.
header-hover-bg String #e1e1e1 Header background color on hover.
rounded Number 8 Border radius in pixels.
shadow Boolean false Adds a shadow effect if present.
blur Boolean false Applies a blur effect if present.
no-auto-close Boolean false Prevents automatic closing when another accordion is opened.

Examples

Custom Background and Text Color

<kittle-accordions title="Custom Styling" color="white" bg-color="blue"> <p>This accordion has a blue background and white text.</p> </kittle-accordions>

Accordion with Blur Effect

<kittle-accordions title="Blur Effect" blur> <p>This accordion has a blurred background effect.</p> </kittle-accordions>

Multiple Accordions with no-auto-close

<kittle-accordions title="First Section" no-auto-close> <p>Content of the first accordion.</p> </kittle-accordions> <kittle-accordions title="Second Section" no-auto-close> <p>Content of the second accordion.</p> </kittle-accordions>
