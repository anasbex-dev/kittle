kittle-navbar is a flexible and customizable navbar component for Kittle UI.

Usage Example

<navbar-kittle bg-dark shadow rounded spacing="30px" logo-img="logo.png" logo-size="50px">
    <a slot="menu" class="nav-link" href="#">Home</a>
    <a slot="menu" class="nav-link" href="#">About</a>
    <a slot="menu" class="nav-link" href="#">Contact</a>
</navbar-kittle>

Attributes & Customization

Attribute Description Default
bg-dark Enables dark mode background. false
bg-blur Adds a blur effect to the navbar background. false
rounded Makes the navbar corners rounded. false
shadow Adds a box-shadow effect. false
bg-color Custom background color (e.g., #ff5722). var(--kittle-light-bg)
text-color Custom text color. var(--kittle-dark-text)
spacing Controls the gap between menu items (e.g., 30px). 20px
logo-img URL of the logo image. None
logo-size Sets the size of the logo image (e.g., 50px). 40px

Styling Example

navbar-kittle {
    --kittle-light-bg: white;
    --kittle-dark-bg: #333;
    --kittle-light-text: black;
    --kittle-dark-text: white;
}

JavaScript API

const navbar = document.querySelector("navbar-kittle");
navbar.setAttribute("bg-color", "#007bff");
navbar.setAttribute("text-color", "#fff");
