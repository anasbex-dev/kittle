Kittle-Text Documentation

Kittle-Text is a flexible custom web component for displaying styled text with various attributes.

Attribute Description Example
type Defines text type (`h1`, `h2`, `p`, etc.) type="h1"
color Text color color="red"
bg-color Background color bg-color="black"
f-sz Font size (px) f-sz="24"
fonts-family Font family fonts-family="Arial"
rounded Border radius (px) rounded="8"
mg Margin mg="10px"
pd Padding pd="5px"
content-position Text alignment (`left`, `center`, `right`) content-position="center"
display Display type (`block`, `inline`, etc.) display="block"
blur Blur effect (px) blur="5"
shadow Text shadow shadow="2px 2px 5px rgba(0,0,0,0.5)"
opacity Opacity (0 - 1) opacity="0.8"

ðŸš€ How to Use

To use the <kittle-text> component, simply include the following in your HTML:

<kittle-text type="h1" color="white" bg-color="black" f-sz="32" 
    fonts-family="Arial" rounded="8" mg="10px" pd="20px" 
    content-position="center" shadow="2px 2px 5px rgba(0,0,0,0.5)" 
    opacity="0.9">
    This is a Custom Heading
</kittle-text>
